Environment Installation with Conda （使用conda配置环境）

Posted on 2021-01-27 Edited on 2021-05-18

Why Conda?

conda 可以用于管理软件包（package），尤其是python的软件包，可以在conda的不同环境（environment）下安装每个环境所需要的package，这样就不会互相干扰，并且易于管理。例如，如果要同时安装python2和python3，就可以在conda下开两个环境分别安装，在调用时启动对应的环境即可。

Anaconda & Miniconda

  • Anaconda包含了一些常用的软件包，而miniconda则只有python和conda；一般可以直接安装miniconda然后再装需要的软件包

Install Miniconda

Miniconda的下载包可以从官网 https://repo.anaconda.com/miniconda/ 找到，里面有不同的版本。这里以下载最新版minconda3为例，其余的可以从上面的网站找到对应的下载方式。如果官网下载太慢，可以尝试从清华镜像https://mirror.tuna.tsinghua.edu.cn/anaconda/miniconda/ 下载，只需要将之前官网的网址完全用这个替代即可。
